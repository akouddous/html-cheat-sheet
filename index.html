HTML Setup

Tag Name Description
doctype <!DOCTYPE HTML> First line of HTML doc. Tells browser I'm writing in HTML
html <html> HTML wrapper
head <head> Doesn't necessarily show on the screen, but controls and defines HTML document
meta <meta> Tells browser character type to show on screen. utf-8 allows for many languages
title <title> Shown in brower tab, and in search results
body <body> Not body of website, but body of HTML document. Anything rendered on screen goes here
